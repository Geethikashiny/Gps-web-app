Install Visual studio code editor
xampp server for mysql database
postman for api testing
nodejs for backend

Running
1. unzip the folder and then in visual studio code open the code folder in the project folder
2. Start the xampp server
3. local host 'http://localhost/phpmyadmin/index.php' create a new database name register
4. import the register.sql file and insert data into database
5. in visual studio code change path to server folder and then click
     npm start
6. then after to run front end change the path to client the click 
     npm start
it will redirects to the webbrowser local host
'http://localhost:3000/'

if necessary install dependencies use 'npm install'
